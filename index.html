Development

Full Project Design and Development

Orange Tentacle can work with you as development specialists as part of a wider team, or can undertake development as part of a wider team. We maintain good relationships with specialist companies in other areas.

Microsoft ASP.NET

Specialists in Microsoft ASP.NET technologies, we can help deliver a range of web applications on the Microsoft stack, and on the wider .NET platform. New projects are all targeted for HTML 5 and designed to make best use of a range of browser environments and technologies.

Modern Tools, Modern Processes

Orange Tentacle believe that staying on and ahead of the curve is important. All of our own projects use the latest tooling, and we project manage using JIRA, an industry leading project management and communication tool. Source code is stored privately in GitHub, and we give our clients access to source code and build instances during development.

Ongoing Maintenance

Orange Tentacle can take projects from other suppliers under our wing and help you with ongoing maintenance and development. Please note that we may not be able to support every project, and we will only be able to offer this service after an inital assessment.

Android Development

A new area for us, we've already dipped our toes in to mobile application development. We have a few applications released to the Google Play store, and plan on releasing more applications in the future. We also have experience with cross-platform mobile development frameworks, for example PhoneGap.

Community Involvement

Orange Tentacle has an ongoing relationship with the tech community. We've been a sponsor of a number of events, spoken at many others, and all our developers contribute back to open source projects.
